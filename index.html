INTRO

「 何かスキルを身につけたい 」

そう思った時に始めたのが、プログラミング・web制作という分野でした。progateや書籍、udemyなどを中心に全て独学で学び、このサイトやWORKに掲載しているアプリケーションを作成しました。スクールには通わなかったため、一つの躓きに対して多くの時間を費やしてしまい、決して効率の良い学習ではなかったかもしれません。しかし、全て自己解決してきたことにより、検索力、自己解決力は少なからず身についたのではないかと思っております。フロントエンドエンジニアになることを、当面の目標にしております。

SKILL

    ProgateでJavaScriptの基礎を学んでから書籍に移行し、初めてのフレームワークとしてVueを選択しました。WORKにVueを用いて、GoogleNEWSのAPIを利用し作成した記事検索アプリを掲載しております。

  • GitHub

    ポートフォリオを掲載するという目的もありますが、業務でも必須となると思い、Udemyの講座にて一連の作業の流れ、エラーが起きた際の対処方法を学びました。

  • PhotoShop

    PhotoShopに関しては、扱えるというほどのレベルではありませんが、コーディングに必要な値の取得方法、画像の書き出しについては理解しています。実際のデザインカンプを用いて学びました。

WORK

記事検索アプリケーション

タイトル : 記事検索アプリ

コード : GitHub

使用した技術・工夫した点 :

  • Vueを用いてGoogleNewsのAPIを利用した記事検索アプリ
  • 入力中、読み込み時、検索結果がない場合など状況に応じたメッセージの表示
  • 一度記事を表示した後にキーワードの再入力を行った場合、表示している記事を非表示にする
  • input要素、selectBox要素がそれぞれ何を入力するところなのかが一目でわかるようにアイコンを設置
  • 一度に何件の記事を取得するか、人気順・新着順のソート機能。入力した内容をAPIのリクエストパラメータに反映させた。
  • 画像を取得する際にNull値で返されることがあるので、その場合は差し替えの画像を表示

MY

19年卒の23歳です。大学時代に周りが就職活動を着々と進めている中、コレと行ってやりたいことも見付からずにいた時に「何かスキルを身に付けたい」と思い始めたのがプログラミング・web制作という分野でした。ゼミの教授が統計学を専門にされている方で、分析を行う際にExcelのVBAを触ったことがあり、Progateで学習を始めた時に基礎の部分はすんなりと理解することができました。初めから職業にしたいと思っていたわけではなかったのでモチベーションを失うことも多々ありました。HTML・CSSから学習を始め、JS、JQuery、PHP、WPなど色々な言語に触れたのですが、どれも中途半端になり、何の為に学習しているのかわからなくもなりました。そんな時にどうしても東京で就職したいと思う出来事がありました。それを叶えるためには何が必要なのかと考えた際に、ポートフォリオサイトの作成やオリジナルアプリの作成をし面接して頂くのが最善の選択だと考えました。先述のように、モチベーションを失いやすいというのが自身の欠点としてあります。しかし、始めに明確な目標、そして、それを達成するのに必要な小さな目標を沢山立てました。それによって進むべき方向がはっきりとして折れずにやってこれました。全て独学で学んできた為、質問すれば5分で解決できるようなことも1,2時間かけていたので決して効率の学習方法ではなかったのかもしれません。ただ、人に質問できないことで自己解決力は少なからず身に付いたとは思っております。冒頭でも述べているようにフロントエンドエンジニアになりたいというのがひとまずの目標なのですが、理由としては、ポートフォリオサイトの制作にあたって様々なサイトを参考にさせて頂いたのですが、自分が魅かれる要素としてアニメーションがカッコいいかどうかというのが大きいです。ファーストインプレッションに興味を持つとサイトの下まで見たい、他にはどんな仕掛けがあるのだろうと考える経験から見た目の部分を作るフロント側になりたいと思いました。
